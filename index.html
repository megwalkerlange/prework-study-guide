HTML

  • HTML(Hypertext Markup Language) is a key programming language in web development. It is used to build the structure of a webpage and provide a framework in which to organise the content for the user
  • The framework is made up by different elements, which all perform a different funtion in structuring the webpage and making the information easier to navigate.
  • The two elements which are the most important are the 'head' and 'body' elements.
  • The head element contains informtaion about the webpage. It is not visible to the user - instead it provides the browser with the required code and information to make the webpage run. This information is called metadata.
  • The body element represents the visible content shown to the user. Developers will use different tags to organise text, headings, lists and images.
