HTML

  • HTML(Hypertext Markup Language) is a key programming language in web development. It is used to build the structure of a webpage and provide a framework in which to organise the content for the user
  • The framework is made up by different elements, which all perform a different funtion in structuring the webpage and making the information easier to navigate.
  • The two elements which are the most important are the 'head' and 'body' elements.
  • The head element contains informtaion about the webpage. It is not visible to the user - instead it provides the browser with the required code and information to make the webpage run. This information is called metadata.
  • The body element represents the visible content shown to the user. Developers will use different tags to organise text, headings, lists and images.

CSS

  • CSS (Cascading Style Sheets) is a coding language that allows developers to add style to a webpage
  • CSS code can be added and embedded into HTML to make visual changes(inline and internal CSS style sheet). However, it is prefereble to create a seperate CSS sheet for the code (external CSS style sheet) and then create a link into the HTML document as it is easier to make changes later on in the project.
  • Example CSS Rule: div {color:blue; text-align: justify;}
  • div = selector = defines the elements and attributes to which the rules and applications apply
  • {colour: blue; = declarations = has two components which are the CSS property we want to the apply and the value of that property. colour is the property and blue is the value.

Git

  • GitHub is an online cloud platform for storing code and development projects. It allows you to showcase your work to others and for developers to work on their projects together at the same time.
  • Work is organised into repositories'b, which store code for projects
  • The most up to date code is stored in the main branch and if developers wish to upload and make changes they must create a new branch. They will work on their code until it is ready to upload, at which poin thtey will upload the altered code and merge the two branches together using a pull request
  • Here is Git commands to upload code to GitHub repo:
  • git add -A
  • git commit -m "description of action"
  • git pull origin main
  • git push origin feature/(repo name here)

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script. In JavaScript the computer always reads the code from top to bottom
